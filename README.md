# WA Geo Data

WA Geo Data is a QGIS plugin that provides access to publicly available map services data published by the Western Australian Department of Mines, Petroleum and Exploration.

## Installation
Plugin can be directly downloaded through the QGIS plugin interface or from the WA Geo Data github repo.

### Requirements
- QGIS version 3.40 or higher
- Internet access
