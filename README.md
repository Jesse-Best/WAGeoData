# ![](/img/wageodata_icon_32.png) WA Geo Data

[WA Geo Data](https://plugins.qgis.org/plugins/WAGeoData/) is a QGIS plugin that provides access to publicly available map services data published by the Western Australian Department of Mines, Petroleum and Exploration. The plugin provides a simple and intuitive interface for discovering and loading geospatial datasets, geology series maps, and related resources available on the Data WA website.

## Features
- Access to DMPE WMS and WFS services
- Predefined QML style definitions for WFS layers
- Full support for Qt5 and Qt6 QGIS versions

## Installation
Plugin can be downloaded through the QGIS plugin interface or from the WA Geo Data GitHub repo.

### Requirements
- Recommended QGIS version 3.40 or higher
- Internet access
